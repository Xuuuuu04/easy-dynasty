# Sound Effects Assets

This folder contains sound effects for the Tarot application.
Detailed below are the collected resource recommendations to enhance the atmosphere.

## Required Files

Please replace the placeholder files with real .wav or .mp3 files (update `SoundContext.tsx` if using mp3).

- `bgm.wav`: Background ambient music (Looping). Suggested: Mystical, Guqin, Ambient.
- `shuffle.wav`: Card shuffling sound.
- `draw.wav`: Card sliding/drawing sound.
- `flip.wav`: Card flipping sound.
- `land.wav`: Card landing on table sound.
- `click.wav`: Soft wood/stone click sound.
- `hover.wav`: Very subtle paper rustle or air whoosh.
- `reveal.wav`: Mystical chime or gong for analysis start.

## Recommended Free Resources

1. **Ambient Music**:
   - Search for "Guqin ambient" or "Mystical drone" on [Freesound.org](https://freesound.org).
   - [YouTube Audio Library](https://www.youtube.com/audiolibrary) (Search: Ambient, Calm, Eastern).

2. **Card Sounds**:
   - "Card Shuffle" on Freesound: https://freesound.org/search/?q=card+shuffle
   - "Card Flip": https://freesound.org/search/?q=card+flip

## Setup

1. Download files.
2. Convert to `.wav` if possible for best compatibility, or `.mp3`.
3. Rename to match the list above.
4. Place in `easydynasty/web/public/sounds/`.
